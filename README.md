Pola Pola Pola
================================
Oatmeal and granola.

An experiment in online interactions. Snap random polaroids with random people.

Hit any URL followed by a `~` to enter a room.

ex:
```
http://?????/~montreal
http://?????/~cats
http://?????/~dogs
http://?????/~js
http://?????/~huh
http://?????/~thiswouldworktoo
http://?????/~anything
```
